# 脑力训练游戏

一个有趣的认知锻炼小游戏，帮助了解自己的短期记忆状况。

## 快速开始

浏览器打开 `outputs/index.html` 即可。

## 玩法

1. **记忆阶段** — 记住 3 个物品（12 秒）
2. **问答❶** — 回答各类趣味题目（约 30 秒）
3. **第一次回忆** — 输入刚才记住的物品
4. **问答❷** — 继续答题（约 3 分钟）
5. **第二次回忆** — 再次回忆物品
6. **结果页** — 评分 + 条形图 + confetti 庆祝

## 功能

- 🧠 **107 道随机题**（计算/常识/自然/历史/社交/趣味）—— 避免连续同选项
- 🌓 **大字模式 / 高对比模式** —— 无障碍适配
- 🔊 **语音朗读**（自动选中文语音，pitch + rate 优化）
- 🎵 **背景音乐**（C 大调合成和弦，可开关）
- 📊 **结果可视化**（得分条形图 + confetti + 音效）
- 📋 **历史记录**（localStorage 存储）
- ⏸ **断点续测**（刷新后可继续）
- 📱 **移动端适配**（320px~480px）

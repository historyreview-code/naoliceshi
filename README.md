# 🧠 脑力训练游戏

短期记忆自测工具。

## 在线体验

访问以下地址即可开始测试：
https://historyreview-code.github.io/naoliceshi/

## 游戏玩法

1. 记住屏幕上的 **3 个物品**
2. 回答简单计算和常识问题
3. **30 秒后**回忆物品
4. 继续回答问题
5. **3 分钟**再次回忆
6. 查看结果

每次测试物品随机更换，避免练习效应。

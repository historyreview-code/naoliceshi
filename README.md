# 🌳 脑力训练游戏 — 短期记忆自测

一个面向老年人的 **短期记忆认知评估工具**，以游戏化的方式帮助初步筛查阿兹海默症相关记忆减损。

**纯前端应用**，无需后端、无需安装、浏览器打开即用。

---

## 测试流程

测试大约 **4 分钟**，分为 6 个阶段：

```
记忆 → 问答❶ → 回忆❶(30秒) → 问答❷ → 回忆❷(3分半) → 结果
```

1. **记忆阶段** — 屏幕展示 3 个日常物品（每次随机从 16 个物品池中抽取），12 秒时间记住
2. **干扰问答** — 随机出计算题和生活常识题，持续答题
3. **第一次回忆（30 秒后）** — 自动弹出回忆页面，输入记住的物品名称
4. **继续干扰** — 继续答题
5. **第二次回忆（3 分 30 秒后）** — 再次回忆物品
6. **结果展示** — 综合评分 + 分层建议

---

## 功能特性

### 核心功能
- **物品随机化** — 16 个物品池每次随机抽取 3 个，避免练习效应
- **模糊匹配** — 输入部分名称也能匹配（如输入"苹"匹配"苹果"）
- **双时间点评估** — 30 秒和 3 分半两个时间点分别测试回忆能力
- **自适应评分** — 综合两次回忆成绩 + 一致性加分，满分 7 分

### 干扰题库（45 题）
- 12 道简单计算（两位数加减法 + 乘法口诀）
- 33 道常识题，覆盖：
  - 生活常识（做饭、出行、健康）
  - 自然科学（动物、植物、气象）
  - 历史文化（节日、地理、传统）
  - 时间认知（月份、星期、日期）

### 适老化设计
- **大字体模式** — 一键切换 22px 基础字号
- **高对比度模式** — 黑底白字，适合视力较弱用户
- **语音朗读** — 记忆和回忆阶段自动中文语音提示
- **声音提醒** — 到达回忆时间点自动播放提示音
- 两种模式设置自动保存到浏览器

### 数据与可靠性
- **进度保护** — sessionStorage 备份，刷新/断网后可继续
- **历史记录** — localStorage 保存最近 50 次测试结果，可追踪趋势
- **结果导出** — 测试结果含具体物品名称和答题统计

## 使用方法

### 直接打开
```bash
# 浏览器打开 index.html 即可
open index.html
```

### 本地服务器（推荐）
```bash
# 使用 Python
python3 -m http.server 8080
# 浏览器访问 http://localhost:8080
```

### 部署到 GitHub Pages
将 `index.html` 推送到 GitHub 仓库，在 Settings → Pages 中启用 `main` 分支即可在线访问。

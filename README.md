# 🌳 脑力训练游戏

一个有趣的认知锻炼小游戏，帮助了解自己的短期记忆状况。

## 快速开始

直接用浏览器打开 `outputs/index.html` 即可。

## 玩法说明

1. **记忆阶段** — 记住屏幕上显示的 **3 个物品**（12秒）
2. **问答阶段❶** — 回答简单计算题、常识题、会话社交题（约30秒）
3. **第一次回忆** — 输入刚才记住的 3 个物品名称
4. **问答阶段❷** — 继续答题（约3分钟）
5. **第二次回忆** — 再次回忆物品
6. **查看结果** — 获得评分、建议和详细得分条形图

## 功能特性

- 🧠 75 道随机题目（计算/常识/自然/历史/社交）
- 🌓 大字模式 / 高对比模式
- 🔊 语音朗读辅助
- 📊 结果页可视化条形图
- 📋 历史记录（本地存储）
- 📱 移动端适配
- ⏸ 断点续测

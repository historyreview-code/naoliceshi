故障排查

如果 curl 报 Failed to connect to 127.0.0.1 port 7890：

# 你的系统有 HTTP 代理，需要临时绕过
unset http_proxy; unset https_proxy
# 或者
http_proxy="" curl -X POST http://localhost:3000/...
